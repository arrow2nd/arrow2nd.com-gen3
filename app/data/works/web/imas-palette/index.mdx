## 概要

アイドルマスターシリーズに登場する、アイドル・ユニットのイメージカラーが検索できる Web アプリ。

## 特徴

名前の一部や色味からイメージカラーを検索し、カラーコードをコピーすることができます。
よく使うカラーを Keep しておくことも。

## 使用技術

Next.js / TypeScript / Tailwind CSS / im@sparql (SPARQL) / Jest / Playwright / Vercel / GitHub Actions

## 制作時期

2021年7月
