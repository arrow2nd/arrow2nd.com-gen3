## 概要

アイドルマスターシリーズに登場するアイドル・スタッフのお誕生日をカウントダウンするサイト。

## ひとこと

Remix と daisyUI を触るついでに、以前突貫で作ったものをリメイクしたものです。

## 使用技術

Remix / TypeScript / Tailwind CSS / daisyUI / im@sparql (SPARQL) / Vercel / Cloudinary

## 制作時期

2023年8月
