## 概要

私のブックマークを検索できる Web アプリ。

## 特徴

リポジトリ上でブックマークを管理していて、Issueを建てることで追加・削除できます。

## ひとこと

異なるブラウザ間でのブックマーク同期が面倒になったのでつくりました。

## 使用技術

Deno / Fresh / Twind / GitHub Actions / Deno Deploy
