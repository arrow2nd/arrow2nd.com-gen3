## 概要

アイドルマスターシャイニーカラーズに登場するアイドル、杜野凛世さんの「お誕生日までの秒数」をカウントダウンするサイト。

## 特徴

動的に生成された OGP 画像とともに、残り秒数を SNS で共有することができます。

## ひとこと

せっかくのお誕生日、何か作りたいな……って思っていたらなんかできていました。Deno Deploy を使ってみたかったというのもあります。

## 使用技術

HTML / CSS / Deno / esbuild / party.js / Deno Deploy / Cloudinary / Github Actions

2021年10月
