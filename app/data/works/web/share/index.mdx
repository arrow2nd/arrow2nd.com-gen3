## 概要

X / Mastodon / Misskey など、各種SNSへのシェアリンクをひとつにまとめられるWebアプリ。

## 特徴

Mastodon / Misskey などは以前共有したサーバーを記憶しておくことができるので、次回以降スムーズに共有できます。

## 使用技術

Next.js / TypeScript / Tailwind CSS / Vercel / Vercel v0

## 制作時期

2023年12月
