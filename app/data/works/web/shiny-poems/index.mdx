## 概要

ゲーム「アイドルマスターシャイニーカラーズ」に登場する、衣装説明の検索ができる Web アプリ。

## 特徴

すきなポエムを SNS 等で共有することができます。

## 使用技術

Next.js / TypeScript / Tailwind CSS / im@sparql (SPARQL) / Jest / Playwright / Vercel / GitHub Actions

## 制作時期

2021年4月
