## 概要

アイマス CD のアートワーク情報を検索できる Web アプリ・Web API

## ひとこと

CDを取り込む際、アートワークをググって持ってくるのが面倒だったのでつくりました。

## 使用技術（フロントエンド）

SolidJS / TypeScript / Milligram / Cloudflare Pages / Vite

## 使用技術（バックエンド）

Deno / TypeScript / hono / MongoDB Atlas / Deno Deploy / GitHub Actions

## 制作時期

2022年4月
