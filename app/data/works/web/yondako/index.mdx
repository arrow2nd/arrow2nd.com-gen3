## 概要

"よんだこと"をわすれないための読書記録サービス。

## ひとこと

積読の管理に読書メーターを使っていたものの、自分には機能が多いな〜と思ったのでつくりました。

## 使用技術

Bun / Next.js / Tailwind CSS / OpenNext / Better Auth / Drizzle ORM / Cloudflare Workers / Cloudflare D1, R2 / NDLサーチ API / biome / Storybook / Figma
