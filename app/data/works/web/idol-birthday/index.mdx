## 概要

アイドルマスターシリーズに登場する全アイドルの「お誕生日までの秒数」をカウントダウンするサイト。

## ひとこと

以前に作った「morino」の全アイドル対応版です。
白菊ほたるさんのお誕生日に間に合うよう突貫で作りました。

## 使用技術

Next.js / TypeScript / Tailwind CSS / im@sparql (SPARQL) / Vercel / Cloudinary / Github Actions

2022年4月
