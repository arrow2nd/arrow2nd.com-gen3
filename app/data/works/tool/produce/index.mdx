## 概要

アイドルマスターシリーズのアイドルのプロフィールを検索することができるLINE Bot

## 特徴

名前の一部や読みがな、誕生日から検索できます。
複数のアイドル名を入力して同時に検索することも可能です。

## 使用技術

Deno / TypeScript / Hono / LINE Messaging API / im@sparql (SPARQL) / Deno Deploy / GitHub Actions

## 制作時期

2020年3月
