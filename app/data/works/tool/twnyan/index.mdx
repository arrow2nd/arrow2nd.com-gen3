## 概要

ターミナル上で動作するシンプルなTwitterクライアント。

## 特徴

ツイートなどの基本的な機能に加えて、マルチアカウント管理や、1分遅れのタイムラインをストリーミングする「疑似 UserStream モード」を搭載しています。

## にゃーん

コマンドラインで twnyan tw と打ち込むだけで「にゃーん」とつぶやけます。

## 使用技術

Go / Twitter API v1.1 / GoReleaser / GitHub Actions

2020年11月
