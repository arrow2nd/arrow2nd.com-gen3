## 概要

ターミナルで動作するTwitterのTUIクライアント。

## 特徴

アンケートやピン留めツイート等の表示が行えます。
タイムラインを自動で更新する疑似Stream Modeもあります。

## 使用技術

Go / Twitter API v2 / GoReleaser / GitHub Actions

## 制作時期

2022年5月
