## 概要

プログラマ向けネーミング生成サービス「codic」の CLI クライアント。

## 特徴

日本語からネーミングを生成することができます。
よく使う命名規則をあらかじめ設定しておくことも可能です。

## 使用技術

Go / codic API / GoReleaser / GitHub Actions

2021年12月
