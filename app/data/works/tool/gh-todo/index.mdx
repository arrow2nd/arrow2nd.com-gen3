## 概要

GitHub Issuesで管理しているToDoをいい感じに操作するためのGitHub CLI拡張コマンド。

## 特徴

fzfを使ってプレビューを表示しつつ、ToDoを絞り込むことができます。

## 使用技術

シェルスクリプト（Bash）
