## 概要

im@sparqlのRDFデータ作成をちょっとだけサポートするVSCode拡張。

## ひとこと

衣装・ユニット・ライブ・ランキングなどの RDF データを対話形式で作成できます。
入力ミスを防ぎつつ、効率よくデータを編集したかったのでつくりました。

## 使用技術

Node.js / TypeScript / webpack / im@sparql (SPARQL) / GitHub Actions

## 制作時期

2022年4月
