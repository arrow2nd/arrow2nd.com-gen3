## 概要

enza対応ゲーム「アイドルマスターシャイニーカラーズ」の非公式専用ブラウザ。

## 特徴

ウィンドウが非アクティブになるのを防ぎ、常時コミュを再生することができます。
1クリックでスクショを保存 & クリップボードへコピーする機能も。

## 使用技術

Electron / React / TypeScript / webpack / Tailwind CSS / GitHub Actions / GitHub REST API

## 制作時期

2021年8月
