## 概要

視聴記録に特化した、アニメ視聴記録サイト「Annict」の非公式CLIクライアント。

## 特徴

作品情報の検索、視聴ステータスの更新、レビューの作成、エピソードの視聴記録ができます。

## 使用技術

Go / Annict Web API (GraphQL) / GoReleaser / GitHub Actions

## 制作時期

2023年2月
