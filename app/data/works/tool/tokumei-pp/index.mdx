## 概要

オモコロのWebラジオを再生する非公式のソフトウェア。

## 特徴

更新が終了しているものを含む、ほぼすべてのオモコロラジオに対応しています。
連続再生・シャッフル再生機能もあります。

## 使用技術

Electron / React / TypeScript / webpack / Tailwind CSS / GitHub Actions / GitHub REST API

## 制作時期

2021年9月
