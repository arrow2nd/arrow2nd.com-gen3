# 0006: Storybook を廃止する

- ステータス: 採用
- 日付: 2026-06-11

## コンテキスト

Next.js 版では `@storybook/nextjs-vite` でコンポーネントカタログを運用していた。Storybook に hono/jsx 用のレンダラーは存在しない。

## 決定

廃止する。stories ファイル（7個）と関連依存を削除した。

## 根拠

- サイト規模（コンポーネント10個強）に対して、dev サーバーでの直接確認で十分と判断
- 維持するなら「dev 限定のプレビュールートを自作」になるが、現時点でその必要性がない

## 再検討の条件

コンポーネント数が増えて単体確認の需要が出たら、dev 限定の `/_preview` ルート（honox のルートを増やすだけ）で代替する。
